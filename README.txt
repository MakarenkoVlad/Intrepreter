    Липовський Клим
    Ляшенко Денис
    Матешко Микола

Програма містить консольний функціонал для збереження та виконання псевдокоду
Можна вводити псевдокод напряму у консолі, або через підгрузку файла з псевдокодом
Файли з псевдокодом можна передавати як з відносним шляхом, так і з абсолютним
Збережні псевдокоди можна виконувати у окремих тредах паралельно
Також для демонстрації роботи тредів реалізована підтримка ітерацій (запустити псведокод у треді N разів)

Скріншоти з прикладами роботи надано у папці /demo

Для запуску програми виконати:
$ python3 main.py
